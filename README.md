# 베이킹 레시피 앱 (프로토타입)

홈베이커용 레시피 기록·스케일 도구. **그램 단위 정확도**가 핵심.
컵/스푼 대신 무게로 기록하고, 배합량이 바뀌면 정확히 다시 계산한다.

## 현재 상태
- 단일 `index.html` 저(低)충실도 프로토타입 (Vanilla JS, 외부 의존성 없음)
- 브라우저에서 `index.html`만 열면 바로 동작 (서버 불필요)
- 데이터는 브라우저 메모리에만 존재 → **새로고침 시 초기화** (영속 저장 미구현)

## 화면
- **리스트**: 레시피 카드 + 검색(이름·제법)
- **기록(편집)**: 제과/제빵 · 제법 · 재료(베이커스% 옵션) · 충전물/토핑(고정 g) ·
  물온도 계산 · 스케일 · 분할(무게/개수/자유성형) · 발효 · 굽기 · 만드는 방법(타임라인)
- **버전**: 저장 시 (날짜+메모) 스냅샷 → 불러오기/삭제
- **후기·사진**: 구운 결과를 사진·메모로 기록

## 핵심 설계 원칙
- 스케일 = 반죽 g 배율(목표÷기준). 베이커스%는 표시 옵션.
- 충전물·토핑 = 고정 g, 스케일 안 함.
- 물온도/실내온도 등 "그날 값"은 레시피 정의(버전)에 저장하지 않음.
- 시드 데이터 = 제과제빵기능사 공개문제 배합 (배합표 수치만 사용, 제법 텍스트는 직접 작성).
